🚀 Create production-ready Express.js APIs in seconds.

Create API is a powerful CLI tool that scaffolds enterprise-grade Express applications with best practices built in. Generate TypeScript or JavaScript projects using MVC or Feature-Based architecture, with optional support for Zod, Joi, Swagger, Pino, Prisma, Drizzle, Mongoose, Husky, Docker, and more.

Features:
• TypeScript & JavaScript support
• MVC and Feature-Based architecture
• Built-in ESLint & Prettier configuration
• Zod, Joi, and Express Validator support
• Swagger API documentation
• Pino logging
• Prisma, Drizzle, and Mongoose integration
• Git & Husky setup
• Docker-ready templates
• Production-ready folder structure

Start your next backend project with a clean, scalable, and maintainable foundation.

npx create-api
